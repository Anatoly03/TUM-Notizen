## Satz von Lagrange
- Ist $(\mathbb{G}, \cdot)$ eine endliche [[Algebraische Strukturen|Gruppe]], und $(\mathbb{U}, \cdot)$ eine Untergruppe dieser Gruppe, so gilt:
$$|\mathbb{U}| \space | \space |\mathbb{G}|$$
## Satz von Euler
- Ist $(\mathbb{G}, \cdot)$ eine endliche Gruppe, so gilt:
$$a^{|\mathbb{G}|} = e$$
